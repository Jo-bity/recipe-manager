# Configure Selected Action in Dedicated Panel

Actions should be added to the Action List first and configured through a dedicated selected-action panel instead of a large inline add form. This keeps Take Image and Unscrewing visible as procedural Actions while giving parameters such as image scope, point cloud, targeting mode, and coordinates enough space for plain-language labels and explanations. The tradeoff is slightly more UI state, but the result is easier for a non-expert technician to understand and extend with future Action types.
