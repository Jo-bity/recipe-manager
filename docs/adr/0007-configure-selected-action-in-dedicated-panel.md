# Configure Selected Step in Dedicated Panel

Steps should be added to the Step List first and configured through a dedicated selected-step panel instead of a large inline add form. The MVP presents Take Image and Unscrewing as Step types backed by one atomic Action per Step, while giving parameters such as image scope, point cloud, targeting mode, and coordinates enough space for plain-language labels and explanations. The tradeoff is slightly more UI state, but the result is easier for a non-expert technician to understand and extend with future compound Steps.
